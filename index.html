အာဖျံကွီး ( မအူပင် )

တာဝန် ပေးတယ်...ဆိုတာ... ပေးအပ်တဲ့ တာဝန် ကို ရဲရဲဝံ့ဝံ့ ကျေပွန်အောင် ထမ်းဆောင်နိုင်သူကို ... တာဝန်ပေးမှ..... မှန်ကန်တဲ့ တာဝန်ပေးအပ်မူ ဖြစ်မှာပါ......။

နှစ်ကြာရှည်စွာ မေ့မျောနေတဲ့ ...နာရီကြီးတောင် ထအော် နေတယ်ဆိုတော့ ... အခြား ပြောစရာ မရှိတော့ပါဘူးးးး ။

ဒီ နာရီစင်နဲ့ပတ်သတ်ပြီး အကြိမ်ကြိမ် ဖေါ်ပြခဲ့ဖူးပါတယ်...။ ခေတ်အဆက်ဆက်... ဆေးလေးသုတ် လိုက် ပြန်အသက်သွင်းလိုက်နဲ့ အသက်ထွက်လု ထွက်ခင် နာရီစင်ဟာ... အားတင်ပြီး ကြုံးရုန်းထပေမယ့် မဟန်တော့တဲ့ အဆုံးးး

ဘယ်အချိန်ကြည့်ကြည့် အချိန်တစ်ခု အမြဲမှန်ဖို့ သူလက်တံတွေ မြှောက်ပြီး ... အသက်ငင်နေရာက အားယူ ဖေါ်ပြ နေရ ရှာပါတယ်....။

ခုတော့.... တက်တက်ကြွကြွ ဝမ်းသာအားရ သူ့အသံတွေ အော်ဟစ်ရင်း .... မြို့ လူထု ကို အချိန်မှန်‌မှန်‌ ဖေါ်ပြနေပါပြီ....။

ကြည်နူးစရာ ပါပဲ... ဝမ်းသာ ဂုဏ်ယူ မိပါတယ်...။

ကျနော်တို့မြို့ဟာ...ဗြုန်းကနဲ့ ..ကြည့်လိုက်ရင် တိုးတတ် ဖွံဖြိုးသာယာတယ်လို့ ထင်ရပေမယ့် ... တကယ်တမ်း သေခြာလေ့လာ ကြည့်ရင်... အများကြီး..လိုအပ်ချက်တွေ ရှိနေသေးပါတယ် ..။

မိုးကလေး တပြိုက် နှစ်ပြိုက် သဲသဲမဲမဲရွာလိုက်တာနဲ့...တစ်ပြိုင်နက် လမ်းပေါ်ရေတွေရောက် နေတာတွေ ....ရေနုတ်မြောင်းမရှိတဲ့အတွက် လျှံတက်လာတဲ့ အညစ်အကြေး တွေဟာ မိုးရေနဲ့အတူ လမ်းမပေါ်ကို ... ရောက်ရှိလာကြပါတော့တယ်...။

အမှိုက်တွေပစ်စရာနေရာမရှိလို့ ကြုံတဲ့ လစ်တဲ့ နေရာမှာပစ်ရင်း ... ကြီးထွားလာတဲ့ အမှိုက်ပုံတွေ က ခွေးဆွဲ.. နွားဆွဲရာက ... အနံ့မျိုးစုံနဲ့ ရူပ်ထွေးလို့ .. လမ်းမပေါ် ဗလပွ ရူပ်ပွလို့ နေကြပြန်တယ်...။

စည်းကမ်းမဲ့ နွားတွေ လမ်းပေါ်အိပ်လို့.... ကား ဆိုင်ကယ်..မောင်းနှင်းသူတွေ အက်စီးဒင့် ဖြစ်ပြီး ဆေးရုံ ရောက် ခဲ့ရတဲ့ အဖြစ်တွေ လည်း ရှိခဲ့ပါတယ်...။

မီးပွိုင့် အပျက်များလို့ ..ယဉ်မောင်းတွေ နားလည်မူလွဲရာက တစ်ယောက်နဲ့တစ်ယောက် မြို့လည်ခေါင်မှာ..အော်ဆဲ ရန်စောင်သံတွေ ... ကလည်းး ကျယ်လောင်နေပြန်တယ်...။

လမ်းဘေးဝဲယာ ခြုံနွယ်ပိတ်ပေါင်းတွေနဲ့ ရေမြောင်း‌ တွေ ပိတ်ဆို့လို့ မြောင်းရေကျန် အကြား ခြင်ပေါက်ဖွားမူများပြီ... "မအူပင် ခြင်ကိုက်တယ်" ဆိုတဲ့ ...စကားကလည်းး ခုချိန်ထိ အမြစ်တွယ် ကျန်ရစ်နေတုံး ပါပဲ....။

စသည်...စသည်..ဖြင့် ....အသေးစား ပျက်ဆီးမူ တွေ အများကြီး ရှိနေပါတယ်...။

ဒီလို အသေးအဖွဲ ဖြစ်ရပ်လေးတွေ က အစ သဘာဝ မကျတဲ့ ကျနော်တို့ မြို့မှာ ... အထာကျတာ တစ်ခုကတော့... မြို ရှောင်လမ်းက လမ်းမီးတိုင်တွေက... မြို့ကို တစ်ညီ တစ်ညွတ်ထဲ ..... .ဦးညွတ် အလေးပြု ( မီးတိုင်တွေ ယိုင်နဲ့ ) နေတာပါပဲ....။

ဒီလို ဖြစ်ရပ်တွေကို တမဟုတ်ခြင်း ပြုပြင်ဖို့ဆိုတာ ..မဖြစ်နိုင်ပါဘူးး။ အချိန်ယူ ပြုပြင်မူတွေနဲ့ ရေရှည် စီမံကိန်းတွေချ အကောင်အထည်ဖေါ်ဆောင်ရွက်မှသာ... အဆင်ပြေပြီးမြောက်မှာပါ ...ကောက်ရိုးမီး လုပ်လို့မရပါဘူးးး။

ဒီလိုအသေးဖွဲ တွေကအစ တိုးတတ် ပြောင်းလဲမူတွေဖြစ်လာဖို့ .... ကျနော်တို့ရှေ့မှာ မားမားမတ်မတ်ရပ်တည်ပြီး ကျနော်တို့ မြို့ဒေသ ဖွံဖြိုးတိုးတတ်ဖို့ အမှန်တကယ် အားထုတ်ကြိုးပမ်းလုပ်‌ ဆောင်မဲ့ သူတော့ တကယ် လိုအပ်‌ နေပါပြီ.....။

ကျနော်.... ကြားခဲ့ရတဲ့ စကား တစ်ခုကို တီးတိုးရည်ရွတ်ရင်... အဖန်ဖန် တလဲလဲ သုံးသပ် စဉ်းစား မိတာလေး .....တင်ပြပါရစေ...။

" ကျနော် ...ပြန်လာတာ...ကိုယ့်ဒေသ ဖွံ့ဖြိုးတိုးတတ်ဖို့ ဘယ်လို လုပ်ပေးရမလဲ ဆိုတဲ့ စိတ်နဲ့ လာတာပါ....…"

ဒီစကားစုကို ... လေ့လာ ကြည့်ရအောင်လားဗျာ....

သူ .... သူ့မြို့ ကို ပြန်ချင်တဲ့ ....ဆန္ဒ ဖြစ်နေတယ်... ။

ဘာအတွက်လဲ.... ကိုယ့် ဒေသ ဖွံဖြိုး တိုးတတ်ဖို့.....။

" သူ့ရဲ့ စိတ်က ...ဘယ်လိုလုပ်ပေးရမလဲ... ဆိုတဲ့..... အာရုံ စူးစိုက်မူ ဖြစ်ပေါ်နေတယ်.... ။"

သူ့ဆီမှာ....ဒေသဖွံဖြိုးတိုးတတ်ဖို့ ... ပြင်းတန်တဲ့ ဆန္ဒ ဖြစ်ပေါ်ပြီး ... ဒါကိုပဲ... အာရုံ စူးစိုက်တဲ့ စိတ် ...နဲ့...အကောင်းဆုံးတွေ ပေးဆပ်ဖို့... အဆင်သင့် ဖြစ်နေပြီ ဆိုတဲ့ သဘော လို့ သုံးသပ်လေ့လာမိပါတယ်....။
